1. From 'Assignment 2 COMP 307', navigate to the directory 'Part 3'

2. Execute the GPProgram using the following command:

java -jar GPClassifier.jar

3. The program will ask you for the name of the training set. Please enter "training.txt", and ensure
that any new training set you may wish to use, is given the same name and is kept in the same directory.

4. The program will now ask you for the name of the test set. Please enter "test.txt", and ensure
that any new test set you may wish to use, is given the same name and is kept in the same directory.

5. The program will now evolve a formula from the training set, use it on the test set, and report the classification accuracy of the best program on both the training and test sets.
